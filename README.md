# About the Project
ELAP is a UI desktop app for macOS. It is made to turn off the built-in monitor of a MacBook when you use an external monitor.

# Background
For many years I worked at laptops using their built-in keyboard as my main keyboard. So the laptop lid does not close. At my first job I loved to work with many screens (up to 3 of them).

One day I decided to use fewer monitors. Just one. My MacBook Air has a 13" screen which is a bit small. But I liked its lightness and battery life. So my solution was to make one external monitor my main screen.

# The Problem
With macOS built-in tools you cannot turn off the built-in screen of a laptop completely when you use an external monitor (or monitors). Reducing the brightness of the built-in screen does not solve this problem completely for two reasons:

- An app window could hide accidentally on the built-in screen and at 0% brightness this is clear and not good
- The screen still draws in the system and this loads the computer

There are programs that let you turn off the built-in screen completely, not just reduce its brightness. But all of them cost money.

# The Solution
Like Torvalds, I saw there are no open source or free solutions for my problem. Not CLI, not with UI. So I tried to use AI (Claude Code) to make a solution for myself.

In the end, to my great surprise, I got a CLI tool that can turn the built-in screen on and off completely!
